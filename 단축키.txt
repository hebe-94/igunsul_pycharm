# venv : 버처 인바이런먼트 파일
# 폴더들 리스트에서 우클릭 하고 show in explor로 하면 파일이 뜨면 됨
부르기 할때는 파이참이 venv를 자동으로 인식하고 자동으로 가져올수 있음
나중에 못가져오면 프로젝트(Day1)폴더를 옮겨서 다시 설정하면된다.

<기능>
1) 창 분리 사용
창을 쪼개서 사용 : (위의 파일 탭)오른쪽 마우스 + split vertically

2) 폰트를 휠로 조절
방법1) shift+shift > change font size로 검색 > switch를 온으로 변경
방법2) File > Settings > General  화면에서 우측 mouse control의 change font size 체크박스 체크

3)
자동주석
(toggle형 이므로 껐다 켰다 할수 있음)
ctrl + /

수동 주석처리
1> 한줄주석 : #
2> 여러줄주석 : ''', """으로 사용


4) 실행
첫번째 실행 : ctrl(파이참에게 이파일을 실행하겠다라는 의미) + shift + F10
두번째 실행(첫번째 실행이후) : shift + F10

출력은
1> 파일이름 (XXXXX~에 있는 파이썬이 YYYY~에 있는 파일을 실행한다는 내용)
2> 실행 요청했던 내용

5) ctrl + d
커서에 있는 내용 1줄 복사

6) 실행 이전, 다음
이전> ctrl + z
다음> ctrl + shift + z

7)
함수 더블클릭 > 마우스 우클릭 > Go To > ctrl + b
def 함수명
기능을 설명 -> 파이썬 인터프리터 속에 있는 기능을 소개
